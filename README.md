# My Portfolio

This is a static portfolio site built with `index.html`, `style.css`, and `script.js`.

## Deploying to Render
1. Sign in to Render and click **New → Static Site**.
2. Connect your GitHub account and select this repository.
3. Configure the form:
   - **Branch:** `main`
   - **Root Directory:** leave blank (the repo root contains `index.html`)
   - **Build Command:** leave blank
   - **Publish Directory:** leave blank
4. Click **Create Static Site** and wait for the initial deploy to complete.

If you prefer automatic creation of the Render site, create a Render API key (Account → API Keys) and paste it in the Render UI or use the Render API.

## Local development
Open `index.html` in a browser or run a simple server:
```
python -m http.server 8000
```
